# Weather-dashboard

## Project setup
```
1. Login in https://www.weatherapi.com/ to get api key
2. Add .env.local file with VUE_APP_URl={url} and VUE_APP_KEY={key}
3. Start npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```
